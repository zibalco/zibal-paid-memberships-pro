Zibal Payment System for Paid Memberships Pro

== Description ==
Iranian Zibal Payment Gateway for the Paid Memberships Pro plugin.

Take payments in IRR through Zibal using the Paid Memberships Pro plugin. Paid Memberships Pro plugin needs to be installed and activated for the Zibal payment gateway to work.

== Installation ==
Upload Plugin to the /wp-content/plugins/ directory
Activate the plugin through the \'Plugins\' menu in WordPress
